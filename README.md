# Презентация: Аналитическая геометрия

## Тема: Уравнения прямой на плоскости

### Выполнил: студент Кутузов Вадим

## Описание проекта

Презентация посвящена изучению различных видов уравнений прямой на плоскости в рамках курса аналитической геометрии. Проект включает теоретический материал, алгоритмы преобразования уравнений и интерактивные программы для работы с уравнениями прямых.

## Содержание презентации:

### Раздел 1: Введение
1. Титульный слайд
2. Определение аналитической геометрии

### Раздел 2: Виды уравнений прямой
3. Уравнение прямой в общем виде
4. Уравнение прямой, проходящей через 2 точки
5. Каноническое уравнение прямой
6. Параметрическое уравнение прямой на плоскости

### Раздел 3: Алгоритмы преобразования
7. Из общего уравнения в каноническое
8. Из общего уравнения в параметрическое
9. Из канонического уравнения в общее

### Раздел 4: JavaScript программы
10. Программа для преобразования уравнений
   - Из общего в каноническое
   - Из общего в параметрическое
   - Из канонического в общее
   - Из канонического в параметрическое
   - Из параметрического в каноническое
   - Из параметрического в общее

### Раздел 5: Графическое построение
11. Программа построения прямой через 2 точки
   - Ввод координат двух точек
   - Вывод уравнения прямой в разных формах
   - Графическое отображение прямой на координатной плоскости

### Раздел 6: Исходный код
12. Исходный код JavaScript программ

### Раздел 7: Дополнительные материалы
13. Забавные иллюстрации и интересные факты
   - Геометрия в нашей жизни

### Раздел 8: Заключение
14. Заключительный слайд с выводами

## Как запустить презентацию:

1. Создайте папку для проекта
2. Скопируйте все 4 файла в эту папку
3. Откройте файл `index.html` в браузере

## Навигация:
- Используйте стрелки на экране
- Клавиши со стрелками на клавиатуре
- Меню в левом верхнем углу

## Функциональность:
1. Навигация по слайдам
2. Интерактивные программы преобразования уравнений
3. Построитель прямой через две точки
4. Пошаговое отображение алгоритмов
5. Адаптивный дизайн для различных устройств
