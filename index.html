Общее уравнение прямой

Общее уравнение:

где A, B, C — постоянные коэффициенты, причем A и B не равны нулю одновременно.

Пример:

Здесь: A = 2, B = -3, C = 6

Особые случаи:

  • Если A = 0 — прямая параллельна оси OX
  • Если B = 0 — прямая параллельна оси OY
  • Если C = 0 — прямая проходит через начало координат

Вектор нормали

Для прямой, заданной общим уравнением Ax + By + C = 0:

\vec{n} = (A, B)

— вектор нормали (перпендикулярен прямой).

Свойства вектора нормали:

• Перпендикулярен прямой

• Определяет ориентацию прямой

• Коэффициенты A и B — его координаты

Пример:

Для прямой 2x - 3y + 6 = 0:

\vec{n} = (2, -3)

Вектор нормали используется для определения угла между прямыми и расстояния от точки до прямой.

Каноническое уравнение прямой

Каноническое уравнение:

где:

  • (x_0, y_0) — точка на прямой
  • (l, m) — координаты направляющего вектора

Пример:

Точка: (2, 1), направляющий вектор: (3, 4)

Вектор направления

Для прямой, заданной каноническим уравнением:

\vec{s} = (l, m)

— направляющий вектор (параллелен прямой).

Свойства:

• Параллелен прямой

• Определяет направление прямой

• Координаты l и m — его компоненты

Пример:

Для прямой \frac{x-2}{3} = \frac{y-1}{4}:

\vec{s} = (3, 4)

Связь с вектором нормали:

\vec{n} \cdot \vec{s} = 0
