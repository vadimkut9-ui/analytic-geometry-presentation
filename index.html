Определение аналитической геометрии

Аналитическая геометрия — раздел математики, в котором геометрические задачи решаются средствами алгебры на основе метода координат.

Основная идея: установление соответствия между геометрическими объектами и уравнениями, описывающими их в координатах.

На плоскости каждая точка определяется парой чисел (x, y), а линии и кривые описываются уравнениями относительно этих координат.

Основные преимущества:

  • Позволяет решать геометрические задачи алгебраическими методами
  • Обеспечивает точность вычислений
  • Дает возможность изучать сложные геометрические фигуры
  • Используется в компьютерной графике, физике, инженерии

Основные понятия аналитической геометрии

Декартова система координат

Система отсчета, состоящая из двух взаимно перпендикулярных осей (X и Y)

\( O(0,0) \) - начало координат
Точка на плоскости

Определяется парой чисел (x, y) — ее координатами

\( M(x, y) \)
Расстояние между точками

Длина отрезка, соединяющего две точки

Уравнение линии

Математическая зависимость между координатами точек линии

\( F(x, y) = 0 \)

Основные задачи аналитической геометрии:

  1. Построение уравнения по геометрическому свойству линии
  2. Исследование геометрических свойств по уравнению
  3. Решение задач на взаимное расположение геометрических объектов
  4. Вычисление расстояний, углов, площадей

Виды уравнений прямой на плоскости

Различные формы записи уравнения одной и той же прямой

Прямая может быть задана:

  • Общим уравнением
  • Через две точки
  • Каноническим уравнением
  • Параметрическими уравнениями

Общее уравнение прямой на плоскости

Общий вид:

где \(A\), \(B\), \(C\) — действительные числа, причем \(A\) и \(B\) не равны нулю одновременно.

Пример:

Уравнение прямой \(2x - 3y + 6 = 0\)

  • Вектор нормали: \(\vec{n} = (A, B) = (2, -3)\)
  • Угловой коэффициент: \(k = -\frac{A}{B} = \frac{2}{3}\)
  • Пересечение с осью OY: при \(x = 0\), \(y = 2\)
  • Пересечение с осью OX: при \(y = 0\), \(x = -3\)

Свойства общего уравнения:

  • Вектор \(\vec{n} = (A, B)\) перпендикулярен прямой
  • Если \(C = 0\), прямая проходит через начало координат
  • Если \(A = 0\), прямая параллельна оси OX
  • Если \(B = 0\), прямая параллельна оси OY

Упрощаем: \(y - 2 = x - 1\) или \(x - y + 1 = 0\)

Алгоритм построения:

  1. Вычислить координаты направляющего вектора: \(\vec{s} = (x_2 - x_1, y_2 - y_1)\)
  2. Записать уравнение в виде: \(\frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1}\)
  3. Преобразовать к общему виду

Каноническое уравнение прямой

Каноническое уравнение:

где \(M_0(x_0, y_0)\) — точка на прямой, а \(\vec{s} = (l, m)\) — направляющий вектор прямой.

Пример:

Прямая, проходящая через точку \(M_0(2, 1)\) с направляющим вектором \(\vec{s} = (3, 4)\):

Направляющий вектор показывает направление прямой, его координаты не должны быть одновременно нулевыми.

Особенности канонического уравнения:

  • Удобно для задания прямой точкой и направлением
  • Легко преобразуется в другие формы
  • Наглядно показывает геометрический смысл
  • Если \(l = 0\): \(\frac{x - x_0}{0} = \frac{y - y_0}{m}\) — вертикальная прямая
  • Если \(m = 0\): \(\frac{x - x_0}{l} = \frac{y - y_0}{0}\) — горизонтальная прямая

Параметрическое уравнение прямой

Параметрическая форма:

где \(t\) — параметр, \(M_0(x_0, y_0)\) — точка на прямой, \(\vec{s} = (l, m)\) — направляющий вектор.

Пример:

Прямая через точку \(M_0(1, -1)\) с направляющим вектором \(\vec{s} = (2, 3)\):

При \(t = 0\) получаем точку \((1, -1)\), при \(t = 1\) — точку \((3, 2)\).

Геометрический смысл параметра:

  • Параметр \(t\) можно интерпретировать как "время"
  • При изменении \(t\) точка "движется" по прямой
  • Направляющий вектор задает скорость движения
  • Каждому значению \(t\) соответствует одна точка на прямой

Алгоритмы преобразования уравнений

Преобразования между различными формами уравнений прямой

Рассмотрим 6 алгоритмов преобразования:

  • Общее → Каноническое
  • Общее → Параметрическое
  • Каноническое → Общее
  • Каноническое → Параметрическое
  • Параметрическое → Каноническое
  • Параметрическое → Общее

Алгоритм: из общего уравнения в каноническое

  1. Найти произвольную точку \(M_0(x_0, y_0)\) на прямой, подставив удобное значение \(x\) (или \(y\)) в уравнение \(Ax + By + C = 0\) и решив относительно другой координаты.
  2. Направляющий вектор: \(\vec{s} = (-B, A)\) или любой коллинеарный ему.
  3. Записать каноническое уравнение: \(\frac{x - x_0}{-B} = \frac{y - y_0}{A}\).

Пример:

\(2x - 3y + 6 = 0\)

  1. При \(x = 0\): \(-3y + 6 = 0 \Rightarrow y = 2\), точка \(M_0(0, 2)\)
  2. \(\vec{s} = (3, 2)\) (можно взять \(\vec{s} = (-B, A) = (3, 2)\))
  3. Каноническое уравнение: \(\frac{x - 0}{3} = \frac{y - 2}{2}\)

Алгоритм: из общего уравнения в параметрическое

  1. Найти точку \(M_0(x_0, y_0)\) на прямой (аналогично предыдущему алгоритму).
  2. Направляющий вектор: \(\vec{s} = (-B, A)\).
  3. Записать параметрическое уравнение: \(\begin{cases} x = x_0 - Bt \\ y = y_0 + At \end{cases}\), где \(t \in \mathbb{R}\).

Пример:

\(2x - 3y + 6 = 0\)

  1. Точка \(M_0(0, 2)\)
  2. \(\vec{s} = (3, 2)\)
  3. Параметрическое уравнение: \(\begin{cases} x = 0 + 3t \\ y = 2 + 2t \end{cases}\)

Алгоритм: из канонического в общее уравнение

  1. Из канонического уравнения \(\frac{x - x_0}{l} = \frac{y - y_0}{m}\) перейти к равенству: \(m(x - x_0) = l(y - y_0)\).
  2. Раскрыть скобки: \(mx - mx_0 = ly - ly_0\).
  3. Перенести все члены в одну сторону: \(mx - ly - mx_0 + ly_0 = 0\).
  4. Обозначить: \(A = m\), \(B = -l\), \(C = -mx_0 + ly_0\).

Пример:

\(\frac{x - 2}{3} = \frac{y - 1}{4}\)

  1. \(4(x - 2) = 3(y - 1)\)
  2. \(4x - 8 = 3y - 3\)
  3. \(4x - 3y - 5 = 0\)

Алгоритм: из канонического в параметрическое

  1. Из канонического уравнения \(\frac{x - x_0}{l} = \frac{y - y_0}{m}\) приравнять каждую дробь к параметру \(t\): \(\frac{x - x_0}{l} = t\) и \(\frac{y - y_0}{m} = t\).
  2. Выразить \(x\) и \(y\) через \(t\): \(\begin{cases} x = x_0 + lt \\ y = y_0 + mt \end{cases}\).

Пример:

\(\frac{x - 2}{3} = \frac{y - 1}{4}\)

  1. Пусть \(\frac{x - 2}{3} = t\) и \(\frac{y - 1}{4} = t\)
  2. Тогда: \(\begin{cases} x = 2 + 3t \\ y = 1 + 4t \end{cases}\)

Алгоритм: из параметрического в каноническое

  1. Из системы \(\begin{cases} x = x_0 + lt \\ y = y_0 + mt \end{cases}\) выразить параметр \(t\) из каждого уравнения: \(t = \frac{x - x_0}{l}\), \(t = \frac{y - y_0}{m}\).
  2. Приравнять полученные выражения: \(\frac{x - x_0}{l} = \frac{y - y_0}{m}\).

Пример:

\(\begin{cases} x = 1 + 2t \\ y = -1 + 3t \end{cases}\)

  1. \(t = \frac{x - 1}{2}\), \(t = \frac{y + 1}{3}\)
  2. Каноническое уравнение: \(\frac{x - 1}{2} = \frac{y + 1}{3}\)

Алгоритм: из параметрического в общее уравнение

  1. Выразить \(t\) из первого уравнения: \(t = \frac{x - x_0}{l}\).
  2. Подставить во второе уравнение: \(y = y_0 + m \cdot \frac{x - x_0}{l}\).
  3. Преобразовать к виду \(Ax + By + C = 0\): \(m(x - x_0) = l(y - y_0) \Rightarrow mx - ly - mx_0 + ly_0 = 0\).

Пример:

\(\begin{cases} x = 1 + 2t \\ y = -1 + 3t \end{cases}\)

  1. \(t = \frac{x - 1}{2}\)
  2. \(y = -1 + 3 \cdot \frac{x - 1}{2}\)
  3. \(2y = -2 + 3x - 3 \Rightarrow 3x - 2y - 5 = 0\)

// Функция преобразования общего уравнения в каноническое
function generalToCanonical(A, B, C) {
    let x0, y0;
    if (Math.abs(A) > 0.0001) {
        y0 = 0;
        x0 = -C / A;
    } else {
        x0 = 0;
        y0 = -C / B;
    }
    
    let l = B;
    let m = -A;
    
    // Упрощаем направляющий вектор
    let gcdVal = gcd(l, m);
    if (Math.abs(gcdVal) > 0.0001) {
        l /= gcdVal;
        m /= gcdVal;
    }
    
    return {x0, y0, l, m};
}

// Функция преобразования общего уравнения в параметрическое
function generalToParametric(A, B, C) {
    const canonical = generalToCanonical(A, B, C);
    return {
        x0: canonical.x0,
        y0: canonical.y0,
        l: canonical.l,
        m: canonical.m
    };
}

// Функция преобразования канонического уравнения в общее
function canonicalToGeneral(x0, y0, l, m) {
    const A = m;
    const B = -l;
    const C = -m*x0 + l*y0;
    return {A, B, C};
}

// Функция преобразования канонического уравнения в параметрическое
function canonicalToParametric(x0, y0, l, m) {
    return {
        x0: x0,
        y0: y0,
        l: l,
        m: m
    };
}

// Функция преобразования параметрического уравнения в каноническое
function parametricToCanonical(x0, y0, l, m) {
    return {
        x0: x0,
        y0: y0,
        l: l,
        m: m
    };
}

// Функция преобразования параметрического уравнения в общее
function parametricToGeneral(x0, y0, l, m) {
    return canonicalToGeneral(x0, y0, l, m);
}
// Функция вычисления уравнения прямой через 2 точки
function lineThroughTwoPoints(x1, y1, x2, y2) {
    // Проверяем, что точки не совпадают
    if (Math.abs(x1 - x2) < 0.001 && Math.abs(y1 - y2) < 0.001) {
        throw new Error("Точки совпадают");
    }
    
    // Вычисляем коэффициенты общего уравнения
    let A = y2 - y1;
    let B = x1 - x2;
    let C = x2 * y1 - x1 * y2;
    
    // Вычисляем каноническое уравнение
    const canonical = {
        x0: x1,
        y0: y1,
        l: x2 - x1,
        m: y2 - y1
    };
    
    // Вычисляем параметрическое уравнение
    const parametric = {
        x0: x1,
        y0: y1,
        l: x2 - x1,
        m: y2 - y1
    };
    
    return {
        general: {A, B, C},
        canonical: canonical,
        parametric: parametric
    };
}

// Функция вычисления точек для построения графика
function calculateLinePoints(A, B, C, xMin = -10, xMax = 10) {
    const points = [];
    
    if (Math.abs(B) > 0.001) {
        // Прямая не вертикальная
        for (let x = xMin; x <= xMax; x += 0.5) {
            let y = (-A * x - C) / B;
            if (y >= -10 && y <= 10) {
                points.push({x, y});
            }
        }
    } else {
        // Вертикальная прямая
        let x = -C / A;
        for (let y = xMin; y <= xMax; y += 0.5) {
            points.push({x, y});
        }
    }
    
    return points;
}

function createLineChart(x1, y1, x2, y2, A, B, C) {

                    ticks: {stepSize: 2},
                    title: {display: true, text: 'Ось X'}
                },
                y: {
                    min: -10,
                    max: 10,

                    ticks: {stepSize: 2},
                    title: {display: true, text: 'Ось Y'}
                }
            },
            plugins: {
                legend: {display: true, position: 'top'},

                    callbacks: {
                        label: function(context) {
                            let label = context.dataset.label || '';
                            if (label) label += ': ';

                                    ', ' + context.parsed.y.toFixed(2) + ')';
                            return label;
                        }
                    }
                }
            }
        }
    };
    
    // Уничтожаем предыдущий график, если есть
    if (window.lineChart) {
        window.lineChart.destroy();
    }

}

Заключение

Итоги изучения уравнений прямой:

  • Прямая на плоскости может быть представлена в различных формах: общее, каноническое, параметрическое уравнения
  • Все формы эквивалентны и могут быть преобразованы друг в друга
  • Каждая форма удобна для решения определенных типов задач
  • Аналитическая геометрия связывает алгебраические уравнения с геометрическими образами

Применение в реальном мире:

  • Компьютерная графика и игровая индустрия
  • Инженерные расчеты и строительство
  • Навигационные системы и картография
  • Машинное обучение и анализ данных
  • Робототехника и автоматизация

Аналитическая геометрия продолжает оставаться фундаментальным инструментом в математике и ее приложениях, демонстрируя красоту связи между алгеброй и геометрией.
