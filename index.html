Раздел 1

Определение аналитической геометрии

Определение аналитической геометрии

Аналитическая геометрия — раздел геометрии, в котором геометрические объекты и их свойства изучаются с помощью алгебраических методов на основе координатного способа.

Основные идеи:

  • Использование системы координат
  • Сопоставление геометрических объектов уравнениям
  • Исследование свойств фигур алгебраическими методами
  • Решение геометрических задач с помощью алгебры

Пример: уравнение прямой на плоскости Ax + By + C = 0

Раздел 2

Виды уравнений прямой на плоскости

1. Уравнение прямой в общем виде

Ax + By + C = 0

где A, B, C — действительные числа, причем A и B не равны нулю одновременно.

Свойства:

  • Вектор нормали: \vec{n} = (A, B)
  • Вектор направления: \vec{s} = (-B, A)
  • Угловой коэффициент: k = -\frac{A}{B} (при B \neq 0)

Пример:

Уравнение: 2x - 3y + 6 = 0

Вектор нормали: \vec{n} = (2, -3)

Вектор направления: \vec{s} = (3, 2)

2. Уравнение прямой, проходящей через 2 точки

\frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1}

где (x_1, y_1) и (x_2, y_2) — координаты двух точек прямой.

Альтернативная форма:

\begin{vmatrix} x - x_1 & y - y_1 \\ x_2 - x_1 & y_2 - y_1 \end{vmatrix} = 0

Пример:

Точки: A(1, 2) и B(4, 5)

Уравнение: \frac{x - 1}{4 - 1} = \frac{y - 2}{5 - 2}

Упрощаем: \frac{x - 1}{3} = \frac{y - 2}{3} \Rightarrow x - 1 = y - 2

Итог: x - y + 1 = 0

3. Каноническое уравнение прямой

\frac{x - x_0}{m} = \frac{y - y_0}{n}

где (x_0, y_0) — точка на прямой, (m, n) — направляющий вектор прямой.

Свойства:

  • Направляющий вектор: \vec{s} = (m, n)
  • Угловой коэффициент: k = \frac{n}{m} (при m \neq 0)
  • Если m = 0 — прямая параллельна оси OY
  • Если n = 0 — прямая параллельна оси OX

Пример:

Точка: (2, 3), направляющий вектор: (4, 5)

Уравнение: \frac{x - 2}{4} = \frac{y - 3}{5}

4. Параметрическое уравнение прямой на плоскости

\begin{cases} x = x_0 + mt \\ y = y_0 + nt \end{cases}

где (x_0, y_0) — точка на прямой, (m, n) — направляющий вектор, t — параметр.

Свойства:

  • Каждому значению t соответствует точка на прямой
  • При t = 0 получаем точку (x_0, y_0)
  • Параметр t может быть любым действительным числом

Пример:

Точка: (1, 1), направляющий вектор: (2, 3)

Уравнение: \begin{cases} x = 1 + 2t \\ y = 1 + 3t \end{cases}

Раздел 3

Алгоритмы преобразования уравнений

Алгоритм: Из общего уравнения в каноническое

Дано: Ax + By + C = 0

  1. Находим любую точку на прямой:
    • Если B \neq 0, полагаем x_0 = 0, тогда y_0 = -\frac{C}{B}
    • Если A \neq 0, полагаем y_0 = 0, тогда x_0 = -\frac{C}{A}
  2. Направляющий вектор: \vec{s} = (-B, A)
  3. Каноническое уравнение: \frac{x - x_0}{-B} = \frac{y - y_0}{A}

Пример:

Дано: 2x - 3y + 6 = 0

1. Точка: при x=0: -3y + 6 = 0 \Rightarrow y = 2, точка (0, 2)

2. Направляющий вектор: \vec{s} = (3, 2)

3. Каноническое уравнение: \frac{x - 0}{3} = \frac{y - 2}{2}

Алгоритм: Из общего уравнения в параметрическое

Дано: Ax + By + C = 0

  1. Находим точку (x_0, y_0) на прямой (как в предыдущем алгоритме)
  2. Направляющий вектор: \vec{s} = (-B, A)
  3. Параметрическое уравнение:
    \begin{cases} x = x_0 - Bt \\ y = y_0 + At \end{cases}

Пример:

Дано: x + 2y - 4 = 0

1. Точка: при x=0: 2y - 4 = 0 \Rightarrow y = 2, точка (0, 2)

2. Направляющий вектор: \vec{s} = (-2, 1)

3. Параметрическое уравнение: \begin{cases} x = 0 - 2t = -2t \\ y = 2 + 1 \cdot t = 2 + t \end{cases}

Алгоритм: Из канонического уравнения в общее

Дано: \frac{x - x_0}{m} = \frac{y - y_0}{n}

  1. Записываем уравнение в виде:
    n(x - x_0) = m(y - y_0)
  2. Раскрываем скобки:
    nx - nx_0 = my - my_0
  3. Переносим все в одну сторону:
    nx - my + (my_0 - nx_0) = 0
  4. Получаем общее уравнение: Ax + By + C = 0, где A = n, B = -m, C = my_0 - nx_0

Пример:

Дано: \frac{x - 1}{2} = \frac{y + 3}{4}

1. 4(x - 1) = 2(y + 3)

2. 4x - 4 = 2y + 6

3. 4x - 2y - 10 = 0

4. Общее уравнение: 4x - 2y - 10 = 0 (или 2x - y - 5 = 0)

// Преобразование из общего уравнения в каноническое
function generalToCanonical(A, B, C) {
    let x0, y0;
    
    // Находим точку на прямой
    if (B !== 0) {
        x0 = 0;
        y0 = -C / B;
    } else {
        y0 = 0;
        x0 = -C / A;
    }
    
    // Направляющий вектор
    const m = -B;
    const n = A;
    
    return {
        canonical: `\\frac{x - ${formatNumber(x0)}}{${formatNumber(m)}} = \\frac{y - ${formatNumber(y0)}}{${formatNumber(n)}}`,
        point: {x: x0, y: y0},
        vector: {m: m, n: n}
    };
}

// Преобразование из канонического в общее
function canonicalToGeneral(x0, y0, m, n) {
    // Общее уравнение: nx - my + (my₀ - nx₀) = 0
    const A = n;
    const B = -m;
    const C = m * y0 - n * x0;
    
    return {
        general: `${formatNumber(A)}x ${formatNumberWithSign(B)}y ${formatNumberWithSign(C)} = 0`,
        coefficients: {A: A, B: B, C: C}
    };
}

// Преобразование из параметрического в каноническое
function parametricToCanonical(x0, y0, m, n) {
    return {
        canonical: `\\frac{x - ${formatNumber(x0)}}{${formatNumber(m)}} = \\frac{y - ${formatNumber(y0)}}{${formatNumber(n)}}`,
        point: {x: x0, y: y0},
        vector: {m: m, n: n}
    };
}
